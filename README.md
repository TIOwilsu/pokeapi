# pokeapp

## Config project
To run project is necessary to use node version `14.17.3` or low, because this project was created a long time ago.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
